README for ShufleServer

1) Install Visual Studio Code 2019
2) Clone the repo
3) Make sure your Node version is at least 4.3. Run "node --version" (without quotes)
3) Run "npm install -g parse-dashboard" (without quotes) from the cloned repo
4) To access the server, run "parse-dashboard --appId shufle --masterKey iLoveRemy911 --serverURL http://shufle.herokuapp.com/parse" (without quotes,
all one line) from the cloned repo
5) Go to localhost:4040 in your browser
